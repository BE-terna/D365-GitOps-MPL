SHORT DESCRIPTION
    D365GitOps module overview.

LONG DESCRIPTION
    D365GitOps provides utilities for Dynamics 365 FO GitOps automation,
    including AxLabel merge-driver registration, label merge/sort operations,
    and daily-build branch preparation for Azure DevOps pull requests.
